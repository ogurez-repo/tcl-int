# tcl-int

## Команды

- `set <name> <value>`
  Создаёт переменную с именем `<name>` и присваивает ей значение `<value>`.

- `put <text>`
  Выводит `<text>` в стандартный поток вывода.

- `put $<name>`
  Выводит значение переменной `<name>`.
  Если переменная не существует, интерпретатор сообщает об ошибке.
